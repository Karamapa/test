Bionic Hand Concept

A planned learning project for designing and documenting a simple bionic hand concept using CAD, Arduino and basic sensor control.

Project goal

The goal is not to claim a finished medical device, but to build a transparent engineering study: mechanical design, simple movement, sensor input and clear documentation of the process.

Planned scope

  • Research basic prosthetic hand mechanisms
  • Create CAD concept parts and assemblies
  • Test Arduino control with simple sensors
  • Document progress with photos, diagrams and short videos
